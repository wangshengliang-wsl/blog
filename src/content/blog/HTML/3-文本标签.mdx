文本是网页最基础的内容。HTML 提供了丰富的文本标签来表达不同类型的文本内容：标题、段落、强调、引用、代码……这篇文章逐一介绍这些标签。

## 标题标签 h1-h6

```html
<h1>一级���题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

HTML 有 6 个级别的标题，`<h1>` 最大最重要，`<h6>` 最小。浏览器默认会给标题加上粗体和不同的字号。

### 标题的语义

标题不只是"大字"，它表示内容的**层级结构**。搜索引擎和屏幕阅读器会根据标题理解页面的组织方式。

想象一本书的目录：

```
第一章  HTML 基础     <- h1
  1.1 什么是 HTML    <- h2
  1.2 HTML 的历史   <- h2
    1.2.1 早期发展  <- h3
    1.2.2 HTML5    <- h3
```

网页结构也应该这样组织：

```html
<h1>HTML 基础教程</h1>

<h2>什么是 HTML</h2>
<p>HTML 是超文本标记语言……</p>

<h2>HTML 的历史</h2>

<h3>早期发展</h3>
<p>HTML 诞生于 1991 年……</p>

<h3>HTML5 时代</h3>
<p>2014 年 HTML5 成为正式标准……</p>
```

### 使用标题的规范

1. **每个页面只用一个 `<h1>`**：`<h1>` 是页面的主标题，应该描述整个页面的主题。博客文章的 `<h1>` 就是文章标题。

2. **不要跳级**：`<h1>` 后面应该是 `<h2>`，不要直接跳到 `<h3>`。

3. **不要用标题来设置字体大小**：想要大字体用 CSS，不要滥用标题标签。

```html
<!-- 错误：用标题来让文字变大 -->
<h1>注意事项</h1>
<h4>请仔细阅读以下内容：</h4>
<p>...</p>

<!-- 正确：用 CSS 控制样式 -->
<h2>注意事项</h2>
<p class="lead">请仔细阅读以下内容：</p>
<p>...</p>
```

## 段落标签 p

```html
<p>这是一个段落。段落是文本内容的基本单位。</p>
<p>这是另一个段落。两个段落之间会有间距。</p>
```

`<p>` 标签定义段落。浏览器会在段落前后自动添加一些外边距（margin），形成视觉上的分隔。

### 段落的特点

- `<p>` 是**块级元素**，独占一行
- 段落内的多个连续空格和换行会被合并成一个空格

```html
<!-- 下面两段显示效果相同 -->
<p>这是 一段 文字。</p>
<p>这是 一段 文字。</p>
```

这是 HTML 的**空白折叠**（whitespace collapsing）规则。如果想保留原始的空格和换行，可以用 `<pre>` 标签或 CSS 的 `white-space` 属性。

### 段落内不能嵌套块级元素

```html
<!-- 错误：p 里面不能放 div -->
<p>
  <div>这是错误的嵌套</div>
</p>

<!-- 正确 -->
<div>
  <p>这是正确的嵌套</p>
</div>
```

浏览器遇到 `<p>` 里的块级元素会自动闭合 `<p>`，导致 HTML 结构混乱。

## 换行与水平线

### 换行 br

```html
<p>
  玫瑰是红色的，<br />
  紫罗兰是蓝色的。
</p>
```

`<br>` 是**空元素**（void element），没有结束标签，也没有内容。它只是在文本中插入一个换行。

注意：不要用多个 `<br>` 来增加段落间距，那是 CSS 的工作。

```html
<!-- 错误：用 br 增加间距 -->
<p>第一段</p>
<br />
<br />
<p>第二段</p>

<!-- 正确：用 CSS -->
<p class="mb-lg">第一段</p>
<p>第二段</p>

<style>
  .mb-lg {
    margin-bottom: 2em;
  }
</style>
```

### 水平线 hr

```html
<p>上面的内容</p>
<hr />
<p>下面的内容</p>
```

`<hr>` 表示段落级别的主题转换，浏览器默认显示为一条水平线。

语义上，`<hr>` 用于分隔两段不同主题的内容，比如小说中的场景切换、文章中的章节分隔。

## 文本格式化标签

### 强调：strong 和 em

```html
<p><strong>重要提示</strong>：请保存好您的密码。</p>
<p>这本书<em>非常</em>值得一读。</p>
```

- `<strong>`：**重要**的文本，浏览器默认显示为粗体
- `<em>`：**强调**的文本，浏览器默认显示为斜体

这两个标签都有语义，屏幕阅读器会用不同的语调朗读。如果只是想要粗体/斜体效果而没有强调的意思，用 CSS：

```html
<!-- 有语义的强调 -->
<p><strong>警告：</strong>此操作不可撤销。</p>

<!-- 纯样式，无语义 -->
<p class="font-bold">商品名称</p>
```

### 粗体和斜体：b 和 i

```html
<p>这是<b>粗体</b>文字。</p>
<p>这是<i>斜体</i>文字。</p>
```

`<b>` 和 `<i>` 曾经是纯样式标签，HTML5 重新定义了它们的语义：

| 标签       | 语义                     | 典型用途             |
| ---------- | ------------------------ | -------------------- |
| `<b>`      | 需要引起注意但不特别重要 | 关键词、产品名       |
| `<i>`      | 不同于周围的文本         | 术语、外语、人物想法 |
| `<strong>` | 重要的内容               | 警告、关键信息       |
| `<em>`     | 强调的内容               | 改变句子的意思       |

实际使用中，如果不确定语义，`<strong>` 和 `<em>` 是更安全的选择。

### 下划线、删除线、高亮

```html
<p>这是<u>带下划线</u>的文字。</p>
<p>原价<s>99 元</s>，现价 79 元。</p>
<p>搜索结果中<mark>高亮显示</mark>关键词。</p>
```

| 标签     | 显示效果 | 语义                               |
| -------- | -------- | ---------------------------------- |
| `<u>`    | 下划线   | 非文本��释（如拼写错误、专有名词） |
| `<s>`    | 删除线   | 不再准确或相关的内容               |
| `<del>`  | 删除线   | 被删除的内容（文档编辑）           |
| `<ins>`  | 下划线   | 新插入的内容（文档编辑）           |
| `<mark>` | 黄色背景 | 高亮/标记                          |

`<del>` 和 `<ins>` 通常成对使用，表示文档的修订：

```html
<p>会议时间：<del>周三下午</del> <ins>周四上午</ins></p>
```

### 上标���下标

```html
<p>H<sub>2</sub>O 是水的化学式。</p>
<p>E = mc<sup>2</sup></p>
<p>这是一个注释<sup>[1]</sup>。</p>
```

- `<sub>`：下标，用于化学式、数学公式
- `<sup>`：上标，用于指数、脚注引用

### 小字

```html
<p>
  <small>版权所有 © 2024</small>
</p>
```

`<small>` 表示附属细则、版权声明、法律条款等小字内容。

## 引用标签

### 块级引用 blockquote

```html
<blockquote>
  <p>学而不思则罔，思而不学则殆。</p>
</blockquote>
```

`<blockquote>` 用于长引用，浏览器默认会添加缩进。可以用 `cite` 属性标明来源：

```html
<blockquote cite="https://example.com/source">
  <p>原始的引用文本……</p>
</blockquote>
<p>—— <cite>《论语》</cite></p>
```

### 行内引用 q

```html
<p>孔子说：<q>学而时习之，不亦说乎？</q></p>
```

`<q>` 用于短引用，浏览器会自动添加引号。

### 引用来源 cite

```html
<p>我正在阅读<cite>《JavaScript 高级程序设计》</cite>。</p>
```

`<cite>` 用于标记作品的标题：书籍、文章、电影、歌曲等。浏览器默认显示为斜体。

## 代码相关标签

### 代码 code

```html
<p>使用 <code>console.log()</code> 输出调试信息。</p>
```

`<code>` 用于标记计算机代码，浏览器默认使用等宽字体显示。

单独的 `<code>` 只能包含一小段代码。对于多行代码块，需要配合 `<pre>` 使用。

### 预格式化文本 pre

```html
<pre>
function greet(name) {
  console.log('Hello, ' + name);
}
</pre>
```

`<pre>` 保留文本中的空格和换行，浏览器默认使用等宽字体。适合展示代码、ASCII 艺术等需要保持格式的内容。

代码块通常这样写：

```html
<pre><code>function add(a, b) {
  return a + b;
}</code></pre>
```

`<pre>` 保留格式，`<code>` 标记语义。

### 键盘输入 kbd

```html
<p>按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 复制。</p>
```

`<kbd>` 表示用户的键盘输入。

### 示例输出 samp

```html
<p>程序输出：<samp>Hello, World!</samp></p>
```

`<samp>` 表示程序或系统的示例输出。

### 变量 var

```html
<p>设 <var>x</var> 为未知数。</p>
```

`<var>` 表示数学或编程中的变量。

## 语义化文本标签

### 缩写 abbr

```html
<p><abbr title="HyperText Markup Language">HTML</abbr> 是网页的基础。</p>
```

`<abbr>` 用于缩写词，`title` 属性提供完整形式。鼠标悬停时会显示完整内容。

### 定义 dfn

```html
<p><dfn>HTML</dfn> 是超文本标记语言的缩写。</p>
```

`<dfn>` 标记正在被定义的术语。

### 时间 time

```html
<p>发布于 <time datetime="2024-01-15">2024年1月15日</time></p>
<p>会议在 <time datetime="14:00">下午两点</time> 开始。</p>
```

`<time>` 标记日期或时间，`datetime` 属性使用机器可读的格式。这有助于搜索引擎理解时间信息。

### 地址 address

```html
<address>
  联系作者：<a href="mailto:author@example.com">author@example.com</a><br />
  地址：北京市朝阳区某某街道
</address>
```

`<address>` 提供联系信息，通常放在 `<article>` 或 `<footer>` 中。

### 双向文本 bdo

```html
<p>正常顺序：Hello</p>
<p>反向顺序：<bdo dir="rtl">Hello</bdo></p>
```

`<bdo>` 覆盖文本的方向，`dir="rtl"` 让文本从右到左显示。

## 分组标签

### div 和 span

```html
<div class="card">
  <h2>标题</h2>
  <p>内容</p>
</div>

<p>这是 <span class="highlight">高亮</span> 文字。</p>
```

- `<div>`：块级容器，用于分组块级元素
- `<span>`：行内容器，用于分组行内元素

这两个标签没有语义，纯粹用于布局和样式。应该优先使用有语义的标签（如 `<article>`、`<section>`），只在没有合适语义标签时才用 `<div>` 和 `<span>`。

## 块级元素与行内元素

HTML 元素分为两大类：

| 类型     | 特点                   | 示例                                |
| -------- | ---------------------- | ----------------------------------- |
| 块级元素 | 独占一行，可设置宽高   | `<div>`、`<p>`、`<h1>`、`<ul>`      |
| 行内元素 | 不换行，宽高由内容决定 | `<span>`、`<a>`、`<strong>`、`<em>` |

```html
<!-- 块级元素独占一行 -->
<div>第一个 div</div>
<div>第二个 div</div>

<!-- 行内元素不换行 -->
<span>第一个 span</span>
<span>第二个 span</span>
```

行内元素不能包含块级元素：

```html
<!-- 错误 -->
<span><div>内容</div></span>

<!-- 正确 -->
<div><span>内容</span></div>
```

## 特殊字符

有些字符在 HTML 中有特殊含义，需要用**实体**（entity）来表示：

| 字符 | 实体名称  | 实体编号  | 说明        |
| ---- | --------- | --------- | ----------- |
| `<`  | `&lt;`    | `&#60;`   | 小于号      |
| `>`  | `&gt;`    | `&#62;`   | 大于号      |
| `&`  | `&amp;`   | `&#38;`   | 和号        |
| `"`  | `&quot;`  | `&#34;`   | 双引号      |
| `'`  | `&apos;`  | `&#39;`   | 单引号      |
|      | `&nbsp;`  | `&#160;`  | 不换行空格  |
| ©   | `&copy;`  | `&#169;`  | 版权符号    |
| ®   | `&reg;`   | `&#174;`  | 注册商标    |
| ™   | `&trade;` | `&#8482;` | 商标        |
| €    | `&euro;`  | `&#8364;` | 欧元        |
| ¥    | `&yen;`   | `&#165;`  | 日元/人民币 |

```html
<p>5 &lt; 10 &amp;&amp; 10 &gt; 5</p>
<p>&copy; 2024 公司名称</p>
```

`&nbsp;` 是不换行空格，可以防止浏览器在该位置换行，也可以用来插入多个连续空格：

```html
<p>价格：&nbsp;&nbsp;&nbsp;99 元</p>
```

## 小结

这篇文章介绍了 HTML 中常用的文本标签：

- **标题**：`<h1>` 到 `<h6>`，注意层级结构
- **段落和换行**：`<p>`、`<br>`、`<hr>`
- **强调和格式**：`<strong>`、`<em>`、`<mark>`、`<del>`、`<ins>`
- **引用**：`<blockquote>`、`<q>`、`<cite>`
- **代码**：`<code>`、`<pre>`、`<kbd>`
- **语义化**：`<abbr>`、`<time>`、`<address>`
- **分组**：`<div>`、`<span>`
