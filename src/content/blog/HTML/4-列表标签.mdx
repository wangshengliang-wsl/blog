列表是组织信息的常用方式。HTML 提供了三种列表：无序列表、有序列表和定义列表。它们各有适用场景，掌握正确的使用方法能让页面结构更清晰。

## 无序列表 ul

无序列表用于列举**没有特定顺序**的项目。

```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>
```

浏览器显示效果：

- 苹果
- 香蕉
- 橙子

### ul 和 li 的关系

- `<ul>` 是无序列表容器（unordered list）
- `<li>` 是列表项（list item）
- `<ul>` 的直接子元素只能是 `<li>`

```html
<!-- 错误：ul 里面直接放文本 -->
<ul>
  苹果 香蕉
</ul>

<!-- 错误：ul 里面直接放 div -->
<ul>
  <div>苹果</div>
  <div>香蕉</div>
</ul>

<!-- 正确 -->
<ul>
  <li>苹果</li>
  <li>香蕉</li>
</ul>
```

`<li>` 里面可以放任何内容：文本、图片、链接，甚至其他列表。

```html
<ul>
  <li>
    <a href="/product/1">
      <img src="apple.jpg" alt="苹果" />
      <span>新鲜苹果</span>
    </a>
  </li>
  <li>
    <a href="/product/2">
      <img src="banana.jpg" alt="香蕉" />
      <span>进口香蕉</span>
    </a>
  </li>
</ul>
```

### 修改列表样式

默认情况下，无序列表会显示实心圆点。可以用 CSS 修改或隐藏：

```css
/* 隐藏列表符号 */
ul {
  list-style: none;
  padding: 0;
  margin: 0;
}

/* 使用不同的符号 */
ul {
  list-style-type: square; /* 方块 */
  list-style-type: circle; /* 空心圆 */
  list-style-type: disc; /* 实心圆（默认） */
}

/* 使用自定义图片 */
ul {
  list-style-image: url('bullet.png');
}
```

### 无序列表的常见用途

无序列表不只是用来显示带圆点的列表。在现代网页中，它被广泛用于：

**导航菜单：**

```html
<nav>
  <ul>
    <li><a href="/">首页</a></li>
    <li><a href="/products">产品</a></li>
    <li><a href="/about">关于</a></li>
    <li><a href="/contact">联系</a></li>
  </ul>
</nav>
```

**标签列表：**

```html
<ul class="tags">
  <li>JavaScript</li>
  <li>HTML</li>
  <li>CSS</li>
</ul>
```

**卡片列表：**

```html
<ul class="card-list">
  <li class="card">
    <h3>文章标题</h3>
    <p>文章摘要...</p>
  </li>
  <li class="card">
    <h3>另一篇文章</h3>
    <p>另一篇摘要...</p>
  </li>
</ul>
```

用 `<ul>` 表达这些内容在语义上是正确的：它们都是"一组相关的项目"。

## 有序列表 ol

有序列表用于列举**有特定顺序**的项目。

```html
<ol>
  <li>打开终端</li>
  <li>输入命令</li>
  <li>按回车执行</li>
</ol>
```

浏览器显示效果：

1. 打开终端
2. 输入命令
3. 按回车执行

### ol 的属性

有序列表有几个实用的属性：

#### type 属性

指定编号类型：

```html
<ol type="1">
  <!-- 1, 2, 3（默认） -->
  <ol type="a">
    <!-- a, b, c -->
    <ol type="A">
      <!-- A, B, C -->
      <ol type="i">
        <!-- i, ii, iii -->
        <ol type="I">
          <!-- I, II, III -->
        </ol>
      </ol>
    </ol>
  </ol>
</ol>
```

示例：

```html
<ol type="A">
  <li>选项 A</li>
  <li>选项 B</li>
  <li>选项 C</li>
</ol>
```

#### start 属性

指定起始编号：

```html
<ol start="5">
  <li>第五步</li>
  <li>第六步</li>
  <li>第七步</li>
</ol>
```

这在分页或继续之前的列表时很有用。

#### reversed 属性

倒序编号：

```html
<ol reversed>
  <li>第三名</li>
  <li>第二名</li>
  <li>第一名</li>
</ol>
```

显示效果：

3. 第三名
4. 第二名
5. 第一名

#### li 的 value 属性

可以给单个列表项指定编号：

```html
<ol>
  <li>第一项</li>
  <li value="10">跳到第十项</li>
  <li>第十一项（自动继续）</li>
</ol>
```

### 有序列表的使用场景

有序列表适用于顺序有意义的内容：

**操作步骤：**

```html
<h3>如何安装 Node.js</h3>
<ol>
  <li>访问 nodejs.org 官网</li>
  <li>下载对应系统的安装包</li>
  <li>运行安装程序</li>
  <li>打开终端，输入 <code>node -v</code> 验证安装</li>
</ol>
```

**排行榜：**

```html
<h3>热门文章排行</h3>
<ol>
  <li>如何学习前端开发</li>
  <li>JavaScript 入门指南</li>
  <li>CSS 布局技巧</li>
</ol>
```

**目录大纲：**

```html
<h3>文章目录</h3>
<ol>
  <li>简介</li>
  <li>基础语法</li>
  <li>进阶技巧</li>
  <li>实战案例</li>
</ol>
```

## 定义列表 dl

定义列表用于展示**术语和定义**的对应关系。

```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，用于构建网页结构。</dd>

  <dt>CSS</dt>
  <dd>层叠样式表，用于控制网页的视觉表现。</dd>

  <dt>JavaScript</dt>
  <dd>脚本语言，用于实现网页的交互功能。</dd>
</dl>
```

### dl、dt、dd 的关系

- `<dl>` 是定义列表容器（description list）
- `<dt>` 是定义的术语（description term）
- `<dd>` 是术语的描述（description details）

一个 `<dt>` 可以对应多个 `<dd>`：

```html
<dl>
  <dt>Web 前端开发</dt>
  <dd>使用 HTML、CSS、JavaScript 构建用户界��</dd>
  <dd>负责网页的视觉呈现和交互体验</dd>
  <dd>需要考虑跨浏览器兼容性</dd>
</dl>
```

多个 `<dt>` 也可以共享一个 `<dd>`：

```html
<dl>
  <dt>HTML</dt>
  <dt>HyperText Markup Language</dt>
  <dd>用于定义网页内容结构的标记语言。</dd>
</dl>
```

### 定义列表的使用场景

虽然名字叫"定义列表"，但它的用途不限于词汇定义。任何"名-值"对都可以用定义列表：

**词汇表/术语解释：**

```html
<h3>术语表</h3>
<dl>
  <dt>API</dt>
  <dd>Application Programming Interface，应用程序编程接口。</dd>

  <dt>DOM</dt>
  <dd>Document Object Model，文档对象模型。</dd>
</dl>
```

**元数据/属性列表：**

```html
<h3>商品信息</h3>
<dl>
  <dt>品牌</dt>
  <dd>Apple</dd>

  <dt>型号</dt>
  <dd>iPhone 15 Pro</dd>

  <dt>颜色</dt>
  <dd>深空黑</dd>

  <dt>存储容量</dt>
  <dd>256GB</dd>
</dl>
```

**FAQ 列表：**

```html
<h3>常见问题</h3>
<dl>
  <dt>如何注册账号？</dt>
  <dd>点击右上角的"注册"按钮，填写邮箱和密码即可完成注册。</dd>

  <dt>忘记密码怎么办？</dt>
  <dd>点击登录页面的"忘记密码"链接，通过邮箱重置密码。</dd>
</dl>
```

**联系信息：**

```html
<dl>
  <dt>地址</dt>
  <dd>北京市朝阳区某某街道 100 号</dd>

  <dt>电话</dt>
  <dd>010-12345678</dd>

  <dt>邮箱</dt>
  <dd>contact@example.com</dd>
</dl>
```

## 嵌套列表

列表可以嵌套使用，形成多级结构。

### 无序列表嵌套

```html
<ul>
  <li>
    前端技术
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>
    后端技术
    <ul>
      <li>Node.js</li>
      <li>Python</li>
      <li>Java</li>
    </ul>
  </li>
</ul>
```

显示效果：

- 前端技术
  - HTML
  - CSS
  - JavaScript
- 后端技术
  - Node.js
  - Python
  - Java

注意嵌套的位置：子列表放在 `<li>` 里面，而不是和 `<li>` 并列。

```html
<!-- 错误：子列表和 li 并列 -->
<ul>
  <li>前端技术</li>
  <ul>
    <li>HTML</li>
  </ul>
</ul>

<!-- 正确：子列表在 li 里面 -->
<ul>
  <li>
    前端技术
    <ul>
      <li>HTML</li>
    </ul>
  </li>
</ul>
```

### 有序列表嵌套

```html
<ol>
  <li>
    准备工作
    <ol>
      <li>安装 Node.js</li>
      <li>安装 VS Code</li>
    </ol>
  </li>
  <li>
    创建项目
    <ol>
      <li>初始化目录</li>
      <li>安装依赖</li>
    </ol>
  </li>
</ol>
```

### 混合嵌套

无序列表和有序列表可以混合嵌套：

```html
<ul>
  <li>
    如何学习前端
    <ol>
      <li>学习 HTML 基础</li>
      <li>学习 CSS 样式</li>
      <li>学习 JavaScript</li>
    </ol>
  </li>
  <li>
    学习资源
    <ul>
      <li>MDN 文档</li>
      <li>在线教程</li>
    </ul>
  </li>
</ul>
```

### 嵌套列表的样式

浏览器默认会给嵌套列表不同的样式：

- 第一层：实心圆
- 第二层：空心圆
- 第三层：方块

可以用 CSS 统一样式：

```css
ul {
  list-style-type: disc;
}

ul ul {
  list-style-type: circle;
}

ul ul ul {
  list-style-type: square;
}
```

## 列表的最佳实践

### 选择正确的列表类型

| 场景     | 推荐列表 |
| -------- | -------- |
| 导航菜单 | `<ul>`   |
| 步骤说明 | `<ol>`   |
| 排行榜   | `<ol>`   |
| 标签云   | `<ul>`   |
| 术语定义 | `<dl>`   |
| 商品属性 | `<dl>`   |
| FAQ      | `<dl>`   |

### 不要滥用列表

```html
<!-- 没必要用列表 -->
<ul>
  <li>只有一项的内容</li>
</ul>

<!-- 直接用段落 -->
<p>只有一项的内容</p>
```

列表是用来组织多个相关项目的，单个项目用段落或其他标签就够了。

### 保持语义正确

```html
<!-- 错误：用列表做布局 -->
<ul>
  <li class="header">头部</li>
  <li class="main">主体</li>
  <li class="footer">底部</li>
</ul>

<!-- 正确：使用语义化标签 -->
<header>头部</header>
<main>主体</main>
<footer>底部</footer>
```

### 无障碍考虑

屏幕阅读器会告诉用户"这是一个列表，包含 X 个项目"。正确使用列表可以帮助视障用户理解页面结构。

导航菜单��用列表时，配合 `<nav>` 标签：

```html
<nav aria-label="主导航">
  <ul>
    <li><a href="/">首页</a></li>
    <li><a href="/products">产品</a></li>
  </ul>
</nav>
```

## 用 CSS 美化列表

### 水平导航菜单

```html
<nav>
  <ul class="nav">
    <li><a href="/">首页</a></li>
    <li><a href="/products">产品</a></li>
    <li><a href="/about">关于</a></li>
  </ul>
</nav>
```

```css
.nav {
  list-style: none;
  padding: 0;
  margin: 0;
  display: flex;
  gap: 20px;
}

.nav a {
  text-decoration: none;
  color: #333;
}

.nav a:hover {
  color: #007bff;
}
```

### 自定义列表符号

使用 `::marker` 伪元素（现代浏览器支持）：

```css
li::marker {
  color: #007bff;
  font-weight: bold;
}
```

使用 `::before` 伪元素（兼容性更好）：

```css
ul {
  list-style: none;
  padding-left: 1.5em;
}

li::before {
  content: '✓';
  color: green;
  margin-right: 0.5em;
}
```

### 带分隔线的列表

```css
ul {
  list-style: none;
  padding: 0;
}

li {
  padding: 10px 0;
  border-bottom: 1px solid #eee;
}

li:last-child {
  border-bottom: none;
}
```

## 小结

这篇文章介绍了 HTML 的三种列表：

- **无序列表 `<ul>`**：用于无特定顺序的项目，如导航菜单、标签
- **有序列表 `<ol>`**：用于有顺序的项目，如步骤、排行榜
- **定义列表 `<dl>`**：用于术语定义、名值对，如商品属性、FAQ

记住：

- `<ul>` 和 `<ol>` 的直接子元素只能是 `<li>`
- 嵌套列表要放在 `<li>` 里面
- 根据内容语义选择合适的列表类型
