### Emmet 快捷方式

Emmet 是 HTML 快速编写的利器：

```
// 基本元素
div → <div></div>
p → <p></p>
a → <a href=""></a>

// 类和 ID
div.container → <div class="container"></div>
div#main → <div id="main"></div>
div.card.active → <div class="card active"></div>

// 嵌套
ul>li → <ul><li></li></ul>
nav>ul>li*3>a →
<nav>
  <ul>
    <li><a href=""></a></li>
    <li><a href=""></a></li>
    <li><a href=""></a></li>
  </ul>
</nav>

// 兄弟
h1+p → <h1></h1><p></p>

// 分组
(header>nav)+main+footer →
<header><nav></nav></header>
<main></main>
<footer></footer>

// 乘法
li*5 → <li></li><li></li><li></li><li></li><li></li>

// 编号
li.item$*3 →
<li class="item1"></li>
<li class="item2"></li>
<li class="item3"></li>

// 文本
p{Hello World} → <p>Hello World</p>

// 属性
a[href="https://example.com" target="_blank"] →
<a href="https://example.com" target="_blank"></a>

// 完整示例
!  → HTML5 文档模板
html:5 → HTML5 文档模板
```

### 代码片段

在 VS Code 中创建自定义代码片段：

```json
// .vscode/html.code-snippets
{
  "HTML5 Template": {
    "prefix": "html5",
    "body": [
      "<!DOCTYPE html>",
      "<html lang=\"zh-Hans\">",
      "<head>",
      "  <meta charset=\"UTF-8\">",
      "  <meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">",
      "  <title>${1:页面标题}</title>",
      "</head>",
      "<body>",
      "  ${0}",
      "</body>",
      "</html>"
    ],
    "description": "HTML5 基础模板"
  },
  "Card Component": {
    "prefix": "card",
    "body": [
      "<article class=\"card\">",
      "  <img src=\"${1:image.jpg}\" alt=\"${2:描述}\">",
      "  <div class=\"card-body\">",
      "    <h3>${3:标题}</h3>",
      "    <p>${4:内容}</p>",
      "  </div>",
      "</article>"
    ],
    "description": "卡片组件"
  }
}
```

## 浏览器开发者工具

### Chrome DevTools

按 F12 或右键检查打开开发者工具。

**Elements 面板**：

- 查看和编辑 HTML
- 实时修改 DOM
- 检查元素样式
- 查看盒模型

```
常用快捷键：
Ctrl/Cmd + Shift + C - 选择元素
Ctrl/Cmd + F - 在 DOM 中搜索
H - 隐藏/显示选中元素
Delete - 删除选中元素
```

**Accessibility 面板**：

- 查看无障碍树
- 检查 ARIA 属性
- 查看对比度
- 检查焦点顺序

**Lighthouse 审计**：

1. 打开 DevTools
2. 选择 Lighthouse 标签
3. 选择类别（Performance, Accessibility, Best Practices, SEO）
4. 点击 Analyze page load

**Network 面板**：

- 查看资源加载
- 分析加载时间
- 检查请求/响应
- 模拟网络条件

### Firefox Developer Tools

Firefox 的开发者工具也很强大：

- **Inspector**：类似 Chrome 的 Elements
- **Accessibility Inspector**：专门的无障碍检查器
- **Network Monitor**：网络监控
- **Style Editor**：样式编辑器

## HTML 验证工具

### W3C Markup Validation

官方的 HTML 验证服务：

```
https://validator.w3.org/

使用方式：
1. 输入 URL
2. 上传文件
3. 直接输入代码
```

### HTMLHint

命令行 HTML 检查工具：

```bash
# 安装
npm install -g htmlhint

# 使用
htmlhint index.html
htmlhint "**/*.html"
```

配置文件 `.htmlhintrc`：

```json
{
  "tagname-lowercase": true,
  "attr-lowercase": true,
  "attr-value-double-quotes": true,
  "doctype-first": true,
  "tag-pair": true,
  "spec-char-escape": true,
  "id-unique": true,
  "src-not-empty": true,
  "attr-no-duplication": true,
  "title-require": true,
  "alt-require": true
}
```

## 开发服务器

### Live Server

VS Code 扩展，自动刷新：

```
安装 Live Server 扩展后：
1. 右键 HTML 文件
2. 选择 "Open with Live Server"
3. 自动在浏览器打开并监听文件变化
```

### Browser Sync

功能更强大的开发服务器：

```bash
# 安装
npm install -g browser-sync

# 启动
browser-sync start --server --files "*.html, css/*.css, js/*.js"

# 代理已有服务器
browser-sync start --proxy "localhost:3000" --files "*.html"
```

### http-server

简单的静态文件服务器：

```bash
# 安装
npm install -g http-server

# 启动
http-server -o

# 指定端口
http-server -p 8080
```

## 代码格式化

### Prettier

流行的代码格式化工具：

```bash
# 安装
npm install -D prettier

# 格式化
npx prettier --write "**/*.html"
```

配置文件 `.prettierrc`：

```json
{
  "printWidth": 80,
  "tabWidth": 2,
  "useTabs": false,
  "semi": false,
  "singleQuote": true,
  "htmlWhitespaceSensitivity": "css"
}
```

### EditorConfig

统一编辑器配置：

```ini
# .editorconfig
root = true

[*]
indent_style = space
indent_size = 2
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

[*.html]
indent_size = 2
```
