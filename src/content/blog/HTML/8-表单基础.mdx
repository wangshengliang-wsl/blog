表单是用户与网站交互的主要方式。登录注册、搜索查询、提交订单……这些功能都离不开表单。这篇文章介绍表单的基础知识。

## form 元素

`<form>` 是表单的容器，包含各种表单控件：

```html
<form action="/login" method="POST">
  <label for="username">用户名：</label>
  <input type="text" id="username" name="username" />

  <label for="password">密码：</label>
  <input type="password" id="password" name="password" />

  <button type="submit">登录</button>
</form>
```

### action 属性

`action` 指定表单提交的目标 URL：

```html
<!-- 提交到指定 URL -->
<form action="/api/submit">
  <!-- 提交到当前页面 -->
  <form action="">
    <!-- 不指定则提交到当前 URL -->
    <form></form>
  </form>
</form>
```

### method 属性

`method` 指定提交方式：

```html
<!-- GET 请求：数据附加在 URL 后面 -->
<form action="/search" method="GET">
  <!-- POST 请求：数据在请求体中 -->
  <form action="/login" method="POST"></form>
</form>
```

| 方式 | 特点                                         | 适用场景       |
| ---- | -------------------------------------------- | -------------- |
| GET  | 数据在 URL 中可见，有长度限制，可被缓存/收藏 | 搜索、筛选     |
| POST | 数据在请求体中，无长度限制，不被缓存         | 登录、提交数据 |

### enctype 属性

`enctype` 指定数据的编码方式，仅在 method="POST" 时有效：

```html
<!-- 默认：URL 编码 -->
<form enctype="application/x-www-form-urlencoded">
  <!-- 文件上传 -->
  <form enctype="multipart/form-data">
    <!-- 纯文本（少用） -->
    <form enctype="text/plain"></form>
  </form>
</form>
```

上传文件必须使用 `multipart/form-data`：

```html
<form action="/upload" method="POST" enctype="multipart/form-data">
  <input type="file" name="avatar" />
  <button type="submit">上传</button>
</form>
```

### target 属性

`target` 指定提交结果在哪里显示，和链接的 target 相同：

```html
<form action="/submit" target="_blank"></form>
```

### novalidate 属性

禁用浏览器的默认验证：

```html
<form novalidate>
  <input type="email" required />
  <button type="submit">提交</button>
</form>
```

## input 元素

`<input>` 是最常用的表单控件，通过 `type` 属性变换成不同的输入类型。

### 文本输入

```html
<!-- 单行文本 -->
<input type="text" name="username" placeholder="请输入用户名" />

<!-- 密码（显示为圆点） -->
<input type="password" name="password" />

<!-- 搜索（可能有清除按钮） -->
<input type="search" name="q" />

<!-- 电话 -->
<input type="tel" name="phone" />

<!-- URL -->
<input type="url" name="website" />

<!-- 邮箱 -->
<input type="email" name="email" />
```

### 数字输入

```html
<!-- 数字 -->
<input type="number" name="age" min="0" max="150" step="1" />

<!-- 范围滑块 -->
<input type="range" name="volume" min="0" max="100" value="50" />
```

number 的属性：

- `min`：最小值
- `max`：最大值
- `step`：步进���（默认 1）

### 日期和时间

```html
<!-- 日期 -->
<input type="date" name="birthday" />

<!-- 时间 -->
<input type="time" name="meeting-time" />

<!-- 日期时间 -->
<input type="datetime-local" name="event-time" />

<!-- 月份 -->
<input type="month" name="start-month" />

<!-- 周 -->
<input type="week" name="week" />
```

### 选择控件

```html
<!-- 单选按钮 -->
<input type="radio" name="gender" value="male" id="male" />
<label for="male">男</label>
<input type="radio" name="gender" value="female" id="female" />
<label for="female">女</label>

<!-- 复选框 -->
<input type="checkbox" name="agree" id="agree" />
<label for="agree">我同意服务条款</label>

<!-- 多个复选框 -->
<input type="checkbox" name="hobbies" value="reading" id="reading" />
<label for="reading">阅读</label>
<input type="checkbox" name="hobbies" value="sports" id="sports" />
<label for="sports">运动</label>
```

同名的 radio 是互斥的（只能选一个），同名的 checkbox 可以多选。

### 文件上传

```html
<!-- 单文件 -->
<input type="file" name="avatar" />

<!-- 多文件 -->
<input type="file" name="photos" multiple />

<!-- 限制文件类型 -->
<input type="file" name="document" accept=".pdf,.doc,.docx" />
<input type="file" name="image" accept="image/*" />
```

`accept` 的值：

- 文件扩展名：`.pdf`, `.jpg`
- MIME 类型：`image/png`, `application/pdf`
- 通配符：`image/*`, `video/*`, `audio/*`

### 颜色选择

```html
<input type="color" name="theme-color" value="#ff0000" />
```

### 隐藏字段

```html
<input type="hidden" name="csrf_token" value="abc123" />
```

用于提交用户不可见的数据，如 CSRF 令牌、ID 等。

### 按钮类型

```html
<!-- 提交按钮 -->
<input type="submit" value="提交" />

<!-- 重置按钮 -->
<input type="reset" value="重置" />

<!-- 普通按钮 -->
<input type="button" value="点击" onclick="handleClick()" />

<!-- 图片按钮 -->
<input type="image" src="submit.png" alt="提交" />
```

推荐使用 `<button>` 代替 `<input type="submit">`，更灵活：

```html
<button type="submit">
  <img src="icon.svg" alt="" />
  提交表单
</button>
```

## input 的通用属性

### name

表单提交时的字段名，必须设置：

```html
<input type="text" name="username" />
```

没有 name 的控件不会被提交。

### value

控件的值：

```html
<!-- 文本输入的默认值 -->
<input type="text" name="city" value="北京" />

<!-- 单选/复选的值 -->
<input type="radio" name="size" value="large" />
```

### placeholder

占位提示文字：

```html
<input type="text" placeholder="请输入用户名" />
```

placeholder 不是 label 的替代品，不要用它代替标签。

### disabled 和 readonly

```html
<!-- 禁用：不能操作，不会提交 -->
<input type="text" name="code" value="ABC" disabled />

<!-- 只读：不能修改，会提交 -->
<input type="text" name="code" value="ABC" readonly />
```

### required

必填字段：

```html
<input type="text" name="username" required />
```

表单提交时会验证必填字段。

### autofocus

页面加载后自动聚焦：

```html
<input type="text" name="search" autofocus />
```

一个页面只应有一个 autofocus。

### autocomplete

控制自动完成：

```html
<!-- 开启自动完成 -->
<input type="text" name="name" autocomplete="name" />

<!-- 关闭自动完成 -->
<input type="password" name="password" autocomplete="off" />

<!-- 特定类型 -->
<input type="email" autocomplete="email" />
<input type="tel" autocomplete="tel" />
```

常用的 autocomplete 值：

- `name`：全名
- `email`：邮箱
- `tel`：电话
- `address-line1`：地址
- `cc-number`：信用卡号
- `new-password`：新密码
- `current-password`：当前密码

### maxlength 和 minlength

```html
<input type="text" name="username" minlength="3" maxlength="20" />
```

### pattern

正则表达式验证：

```html
<!-- 只允许字母和数字 -->
<input type="text" name="code" pattern="[A-Za-z0-9]+" />

<!-- 手机号格式 -->
<input type="tel" name="phone" pattern="1[3-9]\d{9}" />
```

## label 元素

`<label>` 为表单控件提供标签，有两种关联方式：

### 显式关联（推荐）

```html
<label for="username">用户名</label>
<input type="text" id="username" name="username" />
```

`for` 属性的值对应 input 的 `id`。

### 隐式关联

```html
<label>
  用户名
  <input type="text" name="username" />
</label>
```

将 input 放在 label 里面。

### label 的重要性

1. **点击区域扩大**：点击 label 会聚焦到对应的 input
2. **无障碍访问**：屏幕阅读器会朗读 label 内容
3. **语义清晰**：明确字段的用途

```html
<!-- 不好：没有 label -->
<div>用户名：<input type="text" name="username" /></div>

<!-- 好：有 label -->
<div>
  <label for="username">用户名：</label>
  <input type="text" id="username" name="username" />
</div>
```

## button 元素

`<button>` 比 `<input type="button">` 更灵活：

```html
<!-- 提交按钮（默认） -->
<button type="submit">提交</button>

<!-- 重置按钮 -->
<button type="reset">重置</button>

<!-- 普通按钮 -->
<button type="button">点击</button>
```

button 可以包含任何内容：

```html
<button type="submit">
  <svg>...</svg>
  <span>保存</span>
</button>
```

**注意**：在 form 中，button 默认 type 是 submit。如果只是普通按钮，一定要加 `type="button"`：

```html
<form>
  <!-- 会触发表单提交！ -->
  <button onclick="doSomething()">操作</button>

  <!-- 正确：不会触发提交 -->
  <button type="button" onclick="doSomething()">操作</button>
</form>
```

## 表单布局示例

### 垂直布局

```html
<form>
  <div class="form-group">
    <label for="name">姓名</label>
    <input type="text" id="name" name="name" />
  </div>

  <div class="form-group">
    <label for="email">邮箱</label>
    <input type="email" id="email" name="email" />
  </div>

  <button type="submit">提交</button>
</form>
```

```css
.form-group {
  margin-bottom: 1rem;
}

.form-group label {
  display: block;
  margin-bottom: 0.5rem;
}

.form-group input {
  width: 100%;
  padding: 0.5rem;
  border: 1px solid #ccc;
  border-radius: 4px;
}
```

### 水平布局

```html
<form>
  <div class="form-row">
    <label for="name">姓名</label>
    <input type="text" id="name" name="name" />
  </div>

  <div class="form-row">
    <label for="email">邮箱</label>
    <input type="email" id="email" name="email" />
  </div>
</form>
```

```css
.form-row {
  display: flex;
  align-items: center;
  margin-bottom: 1rem;
}

.form-row label {
  width: 100px;
}

.form-row input {
  flex: 1;
}
```

### 登录表单完整示例

```html
<form action="/login" method="POST" class="login-form">
  <h2>用户登录</h2>

  <div class="form-group">
    <label for="username">用户名</label>
    <input
      type="text"
      id="username"
      name="username"
      required
      autocomplete="username"
      placeholder="请输入用户名"
    />
  </div>

  <div class="form-group">
    <label for="password">密码</label>
    <input
      type="password"
      id="password"
      name="password"

```

## 表单提交的工作流程

1. 用户填写表单
2. 点击提交按钮
3. 浏览器进行表单验证（required、pattern 等）
4. 验证通过后，收集所有有 name 的控件的值
5. 根据 method 和 enctype 编码数据
6. 发送请求到 action 指定的 URL
7. 服务器处理请求并返回响应

### JavaScript 提交

可以用 JavaScript 控制表单提交：

```javascript
const form = document.querySelector('form')

form.addEventListener('submit', (e) => {
  e.preventDefault() // 阻止默认提交

  const formData = new FormData(form)

  fetch('/api/submit', {
    method: 'POST',
    body: formData,
  })
    .then((response) => response.json())
    .then((data) => {
      console.log('Success:', data)
    })
})
```
