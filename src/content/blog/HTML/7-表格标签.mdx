表格是展示结构化数据的最佳方式。价格列表、时刻表、统计数据……这些内容用表格呈现最为清晰。这篇文章详细介绍 HTML 表格的各种用法。

## 表格基本结构

一个最简单的表格由 `<table>`、`<tr>`、`<td>` 组成：

```html
<table>
  <tr>
    <td>第一行第一列</td>
    <td>第一行第二列</td>
  </tr>
  <tr>
    <td>第二行第一列</td>
    <td>第二行第二列</td>
  </tr>
</table>
```

- `<table>`：表格容器
- `<tr>`：表格行（table row）
- `<td>`：单元格（table data）

### 表头单元格 th

`<th>` 用于表头单元格，浏览器会默认加粗并居中：

```html
<table>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
    <th>城市</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>25</td>
    <td>北京</td>
  </tr>
  <tr>
    <td>李四</td>
    <td>30</td>
    <td>上海</td>
  </tr>
</table>
```

`<th>` 不只是样式不同，它还有语义作用：告诉浏览器和屏幕阅读器"这是表头"。

### scope 属性

`<th>` 的 `scope` 属性指明它是行标题还是列标题：

```html
<table>
  <tr>
    <th scope="col">产品</th>
    <th scope="col">价格</th>
    <th scope="col">数量</th>
  </tr>
  <tr>
    <th scope="row">苹果</th>
    <td>5 元</td>
    <td>100</td>
  </tr>
  <tr>
    <th scope="row">香蕉</th>
    <td>3 元</td>
    <td>200</td>
  </tr>
</table>
```

- `scope="col"`：列标题
- `scope="row"`：行标题
- `scope="colgroup"`：列组标题
- `scope="rowgroup"`：行组标题

这对无障碍访问非常重要，屏幕阅读器会用正确的方式朗读表格。

## 表格分组

### thead、tbody、tfoot

用这三个标签将表格分为头部、主体和脚部：

```html
<table>
  <thead>
    <tr>
      <th>商品</th>
      <th>单价</th>
      <th>数量</th>
      <th>小计</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>苹果</td>
      <td>5 元</td>
      <td>10</td>
      <td>50 元</td>
    </tr>
    <tr>
      <td>香蕉</td>
      <td>3 元</td>
      <td>20</td>
      <td>60 元</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">总计</td>
      <td>110 元</td>
    </tr>
  </tfoot>
</table>
```

分组的好处：

1. **语义清晰**：结构一目了然
2. **打印分页**：打印长表格时，thead 会在每页重复
3. **样式控制**：方便给不同部分设置不同样式
4. **JavaScript 操作**：更容易定位和操作特定部分

可以有多个 `<tbody>`，但 `<thead>` 和 `<tfoot>` 只能各有一个。

### colgroup 和 col

`<colgroup>` 和 `<col>` 用于定义列组和列的属性：

```html
<table>
  <colgroup>
    <col style="background-color: #f0f0f0" />
    <col span="2" style="background-color: #e0e0e0" />
    <col style="background-color: #d0d0d0" />
  </colgroup>
  <thead>
    <tr>
      <th>商品</th>
      <th>单价</th>
      <th>数量</th>
      <th>小计</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>苹果</td>
      <td>5 元</td>
      <td>10</td>
      <td>50 元</td>
    </tr>
  </tbody>
</table>
```

`<col span="2">` 表示这个 `<col>` 代表两列。

`<colgroup>` 和 `<col>` 主要用于设置整列的样式，可设置的属性有限（主要是 background 和 width）。

## 合并单元格

### colspan 合并列

```html
<table>
  <tr>
    <th colspan="3">学生信息</th>
  </tr>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
    <th>班级</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>18</td>
    <td>一班</td>
  </tr>
</table>
```

`colspan="3"` 让单元格跨越 3 列。

### rowspan 合并行

```html
<table>
  <tr>
    <th>课程</th>
    <th>时间</th>
    <th>教室</th>
  </tr>
  <tr>
    <td rowspan="2">数学</td>
    <td>周一 9:00</td>
    <td>A101</td>
  </tr>
  <tr>
    <td>周三 9:00</td>
    <td>A101</td>
  </tr>
</table>
```

`rowspan="2"` 让单元格跨越 2 行。

### 复杂合并

同时使用 colspan 和 rowspan 可以创建复杂的表格结构：

```html
<table>
  <tr>
    <th colspan="2" rowspan="2"></th>
    <th colspan="2">上半年</th>
    <th colspan="2">下半年</th>
  </tr>
  <tr>
    <th>销量</th>
    <th>收入</th>
    <th>销量</th>
    <th>收入</th>
  </tr>
  <tr>
    <th rowspan="2">北区</th>
    <th>产品A</th>
    <td>100</td>
    <td>1000</td>
    <td>150</td>
    <td>1500</td>
  </tr>
  <tr>
    <th>产品B</th>
    <td>200</td>
    <td>2000</td>
    <td>250</td>
    <td>2500</td>
  </tr>
</table>
```

## caption 表格标题

`<caption>` 为表格提供标题，必须紧跟在 `<table>` 后面：

```html
<table>
  <caption>
    2024年销售数据
  </caption>
  <thead>
    <tr>
      <th>季度</th>
      <th>销售额</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Q1</td>
      <td>100 万</td>
    </tr>
  </tbody>
</table>
```

`<caption>` 默认显示在表格上方居中。可以用 CSS 改变位置：

```css
caption {
  caption-side: bottom; /* 显示在表格下方 */
  text-align: left;
}
```

## 表格样式

浏览器默认的表格样式比较简陋，通常需要用 CSS 美化。

### 基础样式

```css
table {
  border-collapse: collapse; /* 合并边框 */
  width: 100%;
}

th,
td {
  border: 1px solid #ddd;
  padding: 12px 15px;
  text-align: left;
}

th {
  background-color: #f5f5f5;
  font-weight: bold;
}
```

### border-collapse

`border-collapse` 是表格最重要的 CSS 属性：

```css
/* 分离边框（默认） */
table {
  border-collapse: separate;
  border-spacing: 5px; /* 单元格间距 */
}

/* 合并边框（常用） */
table {
  border-collapse: collapse;
}
```

### 斑马条纹

```css
tbody tr:nth-child(odd) {
  background-color: #f9f9f9;
}

tbody tr:nth-child(even) {
  background-color: #ffffff;
}
```

### 悬停高亮

```css
tbody tr:hover {
  background-color: #f0f7ff;
}
```

### 响应式表格

表格在小屏幕上可能超出视口，常见的解决方案：

#### 方案1：水平滚动

```html
<div class="table-container">
  <table>
    ...
  </table>
</div>
```

```css
.table-container {
  overflow-x: auto;
}
```

#### 方案2：堆叠显示

```css
@media (max-width: 768px) {
  table,
  thead,
  tbody,
  th,
  td,
  tr {
    display: block;
  }

  thead {
    display: none;
  }

  td {
    position: relative;
    padding-left: 50%;
  }

  td::before {
    content: attr(data-label);
    position: absolute;
    left: 10px;
    font-weight: bold;
  }
}
```

```html
<tr>
  <td data-label="姓名">张三</td>
  <td data-label="年龄">25</td>
  <td data-label="城市">北京</td>
</tr>
```

## 无障碍访问

### 使用语义化标签

```html
<table>
  <caption>
    员工信息表
  </caption>
  <thead>
    <tr>
      <th scope="col">姓名</th>
      <th scope="col">部门</th>
      <th scope="col">职位</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>张三</td>
      <td>技术部</td>
      <td>工程师</td>
    </tr>
  </tbody>
</table>
```

关键点：

- 使用 `<th>` 而不是样式化的 `<td>`
- 使用 `scope` 属性
- 使用 `<caption>` 或 `aria-label`
- 使用 `<thead>`、`<tbody>`、`<tfoot>`

### 复杂表格的 headers 属性

对于复杂表格，用 `headers` 属性关联单元格和表头：

```html
<table>
  <tr>
    <th id="name">姓名</th>
    <th id="q1">Q1</th>
    <th id="q2">Q2</th>
  </tr>
  <tr>
    <th id="zhang" headers="name">张三</th>
    <td headers="zhang q1">100</td>
    <td headers="zhang q2">150</td>
  </tr>
</table>
```

### aria-describedby

为表格提供额外说明：

```html
<table aria-describedby="table-desc">
  ...
</table>
<p id="table-desc">此表格显示各部门的季度销售数据，单位为万元。</p>
```

## 表格 vs CSS 布局

曾经有段时间，网页布局都用表格来实现。这是错误的做法：

```html
<!-- 错误：用表格做布局 -->
<table>
  <tr>
    <td>侧边栏</td>
    <td>主内容</td>
  </tr>
</table>

<!-- 正确：用 CSS 布局 -->
<div class="layout">
  <aside>侧边栏</aside>
  <main>主内容</main>
</div>
```

表格只应该用于展示**表格数据**，布局应该用 CSS（Flexbox、Grid）。

原因：

1. **语义错误**：表格是数据结构，不是布局工具
2. **无障碍问题**：屏幕阅读器会把布局表格当数据表格朗读
3. **维护困难**：表格布局代码冗长，修改麻烦
4. **响应式困难**：表格不容易适应不同屏幕

## 小结

这篇文章介绍了 HTML 表格的完整用法：

- **基本结构**：`<table>`、`<tr>`、`<td>`、`<th>`
- **表格分组**：`<thead>`���`<tbody>`、`<tfoot>`、`<colgroup>`

- **表格标题**：`<caption>`
- **无障碍访问**：`scope`、`headers` 属性

记住：

- 表格只用于展示表格数据，不要用来布局
- 使用语义化标签和属性提升无障碍体验
- 用 CSS 控制表格样式
