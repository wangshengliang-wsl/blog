## 简介

- **vuedraggable**：列表项拖拽排序
- **vue-drag-resize**：单个元素的拖拽和缩放

## 安装

```bash
npm install vue-drag-resize
```

## 基本用法

```vue
<template>
  <div class="container">
    <VueDragResize
      :w="200"
      :h="150"
      :x="100"
      :y="100"
      :min-width="50"
      :min-height="50"
      @resizing="onResize"
      @dragging="onDrag"
    >
      <div class="content">可拖拽和缩放</div>
    </VueDragResize>
  </div>
</template>

<script setup>
import VueDragResize from 'vue-drag-resize/src'

const onResize = ({ width, height }) => {
  console.log('尺寸:', width, height)
}

const onDrag = ({ left, top }) => {
  console.log('位置:', left, top)
}
</script>

<style scoped>
.container {
  width: 100%;
  height: 500px;
  position: relative;
  background: #f5f5f5;
}
.content {
  width: 100%;
  height: 100%;
  background: #42b983;
  display: flex;
  align-items: center;
  justify-content: center;
  color: white;
}
</style>
```

## 常用属性

| 属性                   | 说明           |
| ---------------------- | -------------- |
| w / h                  | 初始宽高       |
| x / y                  | 初始位置       |
| min-width / min-height | 最小尺寸       |
| max-width / max-height | 最大尺寸       |
| is-draggable           | 是否可拖拽     |
| is-resizable           | 是否可缩放     |
| aspect-ratio           | 保持宽高比     |
| parent-limitation      | 限制在父元素内 |

## 常用事件

| 事件        | 参数              | 说明     |
| ----------- | ----------------- | -------- |
| @resizing   | { width, height } | 缩放中   |
| @resizestop | { width, height } | 缩放结束 |
| @dragging   | { left, top }     | 拖拽中   |
| @dragstop   | { left, top }     | 拖拽结束 |

## 应用场景

- 可视化编辑器
- 图片裁剪
- 仪表盘布局
- 自由布局设计器
