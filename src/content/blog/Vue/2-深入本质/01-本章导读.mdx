🎯 **本章目标**：理解 Vue 最核心的原理，从"会用"到"懂原理"。

## 为什么要学原理

1. **方案设计**：知道底层机制，才能做出正确的技术决策
2. **问题排查**：遇到奇怪的 bug，能从原理角度分析
3. **面试必考**：Vue 原理是前端面试的高频考点
4. **进阶基石**：阅读源码、贡献开源的前提

## 本章内容概览

| 主题       | 内容                        |
| ---------- | --------------------------- |
| 虚拟 DOM   | 本质、作用、diff 算法       |
| 模板编译   | 编译过程、渲染函数          |
| 响应式系统 | Proxy、依赖收集、派发更新   |
| 组件机制   | 生命周期、Props、插槽的本质 |
| 内置组件   | KeepAlive、Teleport 原理    |

## 学习建议

🔶 **按顺序学习**：每篇内容都依赖前面的知识

🔶 **动手验证**：看完原理后，自己写代码验证

🔶 **多看几遍**：第一遍看不懂很正常，多看几遍会豁然开朗

🔶 **结合源码**：有余力的话，对照 Vue 源码阅读
