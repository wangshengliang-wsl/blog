## 面试题分类

### 使用相关

- Vue 中如何自定义指令？
- 说一下 Vue 中的生命周期？
- Vue 中组件通信的方式有哪些？
- v-if 和 v-show 的区别？
- 常见的修饰符有哪些？
- ....

### 原理相关

**非八股文**（影响实际开发）：

- Vue 响应式渲染原理
- 虚拟 DOM 是什么？
- key 有什么作用？

**八股文**（不懂也不影响开发）：

- diff 算法原理
- Vue2 和 Vue3 的 diff 区别

## 本系列内容

主要讲解典型、有代表性的面试题目：

1. Vue3 整体变化
2. 响应式原理变化
3. nextTick 实现原理
4. Vue 运行机制
5. 渲染器核心功能
6. diff 算法详解
7. 模板编译器
8. 性能优化
