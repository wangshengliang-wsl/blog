| 模块       | 内容                                             |
| ---------- | ------------------------------------------------ |
| Vue 进阶   | 属性透传、依赖注入、组合式函数、自定义指令、插件 |

| Pinia      | 自定义插件、通信方式总结                         |
| 性能优化   | 懒加载、虚拟列表、异步组件                       |
| 第三方库   | VueUse、VueDraggable、VueChartJS 等              |
| Vite       | 配置、插件、环境变量、构建优化                   |
