## 简介

VueUse 是基于 Composition API 的工具库，提供大量实用的组合式函数。

官网：https://vueuse.org/

## 安装

```bash
npm install @vueuse/core
```

## 常用函数

### 状态管理

```js
import { useToggle, useCounter } from '@vueuse/core'

// 开关状态
const [isOpen, toggle] = useToggle()
toggle() // 切换状态

// 计数器
const { count, inc, dec, set, reset } = useCounter(0)
```

### 浏览器 API

```js
import { useLocalStorage, useClipboard, useFetch } from '@vueuse/core'

// localStorage 响应式
const name = useLocalStorage('name', 'default')

// 剪贴板
const { copy, copied } = useClipboard()
await copy('Hello')

// Fetch 请求
const { data, error, isFetching } = useFetch('/api/users')
```

### 传感器

```js
import { useMouse, useGeolocation, useWindowSize } from '@vueuse/core'

// 鼠标位置
const { x, y } = useMouse()

// 地理位置
const { coords } = useGeolocation()

// 窗口尺寸
const { width, height } = useWindowSize()
```

### UI 工具

```js
import { useDark, useFullscreen } from '@vueuse/core'

// 暗色模式
const isDark = useDark()
isDark.value = true

// 全屏
const { isFullscreen, enter, exit, toggle } = useFullscreen()
```

### 工具函数

```js
import { useDebounceFn, useThrottleFn } from '@vueuse/core'

// 防抖函数
const debouncedFn = useDebounceFn(() => {
  // 处理逻辑
}, 500)

// 节流函数
const throttledFn = useThrottleFn(() => {
  // 处理逻辑
}, 500)
```

## 完整示例

```vue
<script setup>
import { useMouse, useWindowSize, useLocalStorage } from '@vueuse/core'

const { x, y } = useMouse()
const { width, height } = useWindowSize()
const theme = useLocalStorage('theme', 'light')
</script>

<template>
  <div>
    <p>鼠标: {{ x }}, {{ y }}</p>
    <p>窗口: {{ width }} x {{ height }}</p>
    <button @click="theme = theme === 'light' ? 'dark' : 'light'">
      切换主题: {{ theme }}
    </button>
  </div>
</template>
```
