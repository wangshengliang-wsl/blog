# 你好，我是 wsl

一个喜欢琢磨代码的前端工程师，目前在做 Web 开发相关的工作。

## 关于我

写代码这件事吧，对我来说就像玩解谜游戏——每天都有新的小难题冒出来，上午可能还在跟页面布局死磕，下午又开始琢磨怎么让代码跑得更快，晚上忍不住又去研究点新工具新方法。说实话，每次把一个棘手的问题解决掉，那种感觉真的挺上头的。

这些年干过的活儿还挺杂的：从零开始搭过电商网站，重构过那种让人头大的管理系统，也帮团队搭过不少提效的工具。我这个人有点较真，为了让一个动画效果更顺滑，能反复调试到半夜；为了让网页加载快那么一点点，愿意花好几天去研究优化方案。同事都说我有点"偏执"，不过我觉得吧，做东西就得做到自己满意，用户体验这事儿马虎不得。

## 为什么写博客

其实建这个博客的想法挺简单的。一方面呢，是想把平时工作中遇到的问题、踩过的坑都记下来，省得以后再碰到同样的问题还得从头想；另一方面，如果我写的这些东西刚好能帮到正在头疼同样问题的朋友，那就更好了。

我一直觉得，做技术这行没有什么"标准答案"，大家的方案各有各的道理，适合自己项目的就是好方案。所以这里也欢迎大家来聊聊，有不同看法尽管说，咱们一起讨论讨论。

## 工作之外

除了写代码，我平时还挺喜欢捣鼓点别的。比如研究怎么把歌唱得好听一点，练练音感，弹弹钢琴。游戏偶尔也打，主要是放松。还有就是看书和纪录片，技术类的、人文类的都看，有时候同一本书隔段时间再翻，总能看出点新东西来。

怎么说呢，我觉得生活里得有点代码之外的东西，这样反而写起代码来更有状态，脑子也更灵活。
